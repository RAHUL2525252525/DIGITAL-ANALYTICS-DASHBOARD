# Aether Sovereign Digital Analytics Dashboard

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Place your `serviceAccountKey.json` in the root directory.
3. Update `firebase-config.js` if necessary.
4. Run the app: `python app.py`

## Features
- **Neon Glass Theme**: Premium SaaS aesthetic across all pages.
- **Auth Redirection Fix**: Smart JS listeners ensure no users are stuck on login.
- **Real-Time Data**: Firestore integration for live user management.
- **Admin SDK Control**: Secure backend user deletion.
